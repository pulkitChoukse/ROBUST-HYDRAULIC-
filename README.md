# ROBUST-HYDRAULIC-
Hydraulic transients, or water hammer, occur when water flow in penstocks changes suddenly, leading to sharp pressure surges. In hydropower systems, improper guide vane closure can cause equipment damage or instability.

This project develops a simplified one-dimensional simulation model to study pressure and velocity variations during gate closure. It identifies safe closure times, generates pressure/velocity graphs, and integrates a basic machine learning model for quick predictions. A Python-based GUI allows users to input parameters and visualize results.

Key Features:

Simplified penstock simulation using fluid mechanics.
Closure time analysis with safe operating range identification.
Optional ML predictions for closure time.
User-friendly desktop interface with real-time graphs.

Technologies: Python, NumPy, Matplotlib, Scikit-learn, Tkinter.
Applications: Educational demonstrations, basic analysis, and project presentations.
Outcome: A practical tool combining simulation, visualization, and ML to make hydraulic transient analysis accessible and effective.
